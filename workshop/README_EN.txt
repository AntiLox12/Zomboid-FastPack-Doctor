STEAM WORKSHOP UPLOAD FOLDER

1. Copy the ZomboidFastPackDoctor folder to:
   %USERPROFILE%\Zomboid\Workshop\

2. Start Project Zomboid Build 42.
3. Open Workshop -> Create and update items.
4. Select ZomboidFastPackDoctor.
5. Review the title, description, preview.png, and visibility.
6. Select suitable tags in the uploader and press Upload.

On first publication Steam adds id=<Workshop ID> to workshop.txt. Keep that
line for later updates.

After publication, a regular user subscribes, enables the mod, loads a world,
and opens Pause Menu -> active mod list -> FastPack Report.

The full Companion is a separate download:
https://github.com/AntiLox12/Zomboid-FastPack-Doctor/releases/latest
