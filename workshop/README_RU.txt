ГОТОВАЯ ПАПКА ДЛЯ STEAM WORKSHOP

1. Скопируйте папку ZomboidFastPackDoctor в:
   %USERPROFILE%\Zomboid\Workshop\

2. Запустите Project Zomboid Build 42.
3. Откройте Workshop -> Create and update items.
4. Выберите ZomboidFastPackDoctor.
5. Проверьте заголовок, описание, preview.png и видимость.
6. Выберите подходящие теги в загрузчике и нажмите Upload.

При первом создании Steam добавит строку id=<Workshop ID> в workshop.txt.
Не удаляйте её при последующих обновлениях.

После публикации обычный пользователь:
- нажимает Subscribe;
- включает мод в менеджере B42;
- загружает мир;
- открывает Pause Menu -> список активных модов -> Отчёт FastPack.

Полный Companion скачивается отдельно:
https://github.com/AntiLox12/Zomboid-FastPack-Doctor/releases/latest
