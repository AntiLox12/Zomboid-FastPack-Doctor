[h1]Zomboid FastPack Doctor[/h1]

Runtime-диагностика и API ленивой инициализации для больших сборок
Project Zomboid Build 42.

[h2]Как пользоваться[/h2]

1. Подпишитесь и включите Zomboid FastPack Doctor.
2. Загрузите мир.
3. Откройте меню паузы, список активных модов и нажмите «Отчёт FastPack».
4. Кнопка GitHub откроет полный Companion для проверки до запуска игры.

[h2]Мод Workshop и Companion[/h2]

Подписка Workshop устанавливает только внутриигровую часть. Для проверки
размеров модов, дубликатов item/vehicle/recipe ID, пересечений карт,
зависимостей, ошибок console.txt, устаревшей структуры, safe-mode профилей и
HTML-отчётов нужен бесплатный Windows Companion:

https://github.com/AntiLox12/Zomboid-FastPack-Doctor/releases/latest

[h2]Важное ограничение[/h2]

FastPack Doctor не переписывает нативный загрузчик и не обещает выдуманное
ускорение в процентах. Runtime-профилирование охватывает совместимые
обработчики, зарегистрированные через API FastPack.
